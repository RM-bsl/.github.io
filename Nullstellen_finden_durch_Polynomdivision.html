Detaillierte Zwischenschritte:

1. Zyklus (x hoch 2)

  • Dividieren: x hoch 3 : x = x hoch 2
  • Rückmultiplizieren: x hoch 2 * (x - 2) = x hoch 3 - 2x hoch 2
  • Subtraktion: -4x hoch 2 - (-2x hoch 2) = -2x hoch 2

2. Zyklus (-2x)

  • Dividieren: -2x hoch 2 : x = -2x
  • Rückmultiplizieren: -2x * (x - 2) = -2x hoch 2 + 4x
  • Subtraktion: x - (4x) = -3x

3. Zyklus (-3)

  • Dividieren: -3x : x = -3
  • Rückmultiplizieren: -3 * (x - 2) = -3x + 6
  • Subtraktion: 6 - 6 = 0
