Die Polynomdivision

Gradreduzierung ganzrationaler Funktionen strukturiert erarbeiten. Visualisiert nach den UI/UX-Prinzipien des Microsoft Fluent Design Systems.

Navigation
Für quadratische Gleichungen (ax² + bx + c = 0) besitzen wir zuverlässige Formeln wie die pq-Formel oder Mitternachtsformel. Bei Gleichungen ab dem 3. Grad (x³, x⁴, etc.) fehlen diese. Durch die Polynomdivision spalten wir einen bekannten Faktor ab und reduzieren den Exponenten um eine Stufe nach unten, damit die bekannten Verfahren wieder greifen. Dies wird auch als Gradreduktion bezeichnet.
Sie ist das Werkzeug der letzten Wahl. Wir nutzen sie exakt dann, wenn einfachere Verfahren scheitern: Ausklammern klappt nicht, weil ein konstantes Glied ohne x existiert (wie die +6 im Beispiel). Substitution scheitert ebenfalls, weil die Exponenten ungerade und gemischt vorliegen (z.B. x³ und x²).

Beispielfunktion: f(x) = x³ - 4x² + x + 6

Schritt-für-Schritt Algorithmus

Um ein Polynom dritten Grades zu reduzieren, benötigen wir eine erste ganzzahlige Nullstelle. Diese "erraten" wir durch systematisches Testen der Teiler des Absolutglieds (+6), also ±1, ±2, ±3, ±6. Wir prüfen x = 2:

f(2) = 2³ - 4 · 2² + 2 + 6 = 8 - 16 + 2 + 6 = 0
Schul-Praxistipp: Systematisches Testen per Taschenrechner

Statt jeden Teiler mühsam von Hand in die Formel einzusetzen, kannst du die Wertetabelle (TABLE-Funktion) deines wissenschaftlichen Taschenrechners nutzen. Gib dort einfach den Funktionsterm ein, wähle als Startwert -6, als Endwert 6 und als Schrittweite (Step) 1. Der Taschenrechner zeigt dir nun alle Funktionswerte auf einen glance. Du musst in der Tabelle nur noch nach dem Eintrag suchen, bei dem der Funktionswert genau 0 beträgt.

Strukturwissen: Da das Ergebnis f(2) = 0 ist, wissen wir nach dem Linearfaktorsatz, dass sich das Polynom ohne Rest durch den Term (x - 2) teilen lässt.

Wir betrachten ausschließlich den vordersten Term des Dividenden (x³) und teilen diesen durch das führende Glied des Divisors (x).

( x³ - 4x² + x + 6 ) : ( x - 2 ) = x²
1. Dividieren:
x³ : x = x²
2. Rückmultiplizieren:
x² · (x - 2) = x³ - 2x²

Der zurückmultiplizierte Term wird vom Dividenden abgezogen. Die Tabellenlinie markiert das Ergebnis dieser Subtraktion. Danach holen wir das nächste Glied (+ x) herunter.

( x³ - 4x² + x + 6 ) : (x - 2) = x²
- ( x³ - 2x² )
- 2x² + x
⚠️ Vorzeichen-Falle (Minusklammer):
Wir rechnen spaltenweise:
Für die x³-Spalte: x³ - x³ = 0.
Für die x²-Spalte: -4x² - (-2x²) = -4x² + 2x² = -2x². Achte darauf, dass sich das Vorzeichen in der Klammer umdreht!

Wir wiederholen die drei Grundschritte (Dividieren, Multiplizieren, Subtrahieren) mit unserem neuen Restpolynom -2x² + x.

( x³ - 4x² + x + 6 ) : (x - 2) = x² - 2x
- ( x³ - 2x² )
- 2x² + x
- ( - 2x² + 4x )
- 3x + 6

Ein letztes Mal dividieren wir den verbleibenden Rest -3x + 6 durch x - 2.

( x³ - 4x² + x + 6 ) : (x - 2) = x² - 2x - 3
- ( x³ - 2x² )
- 2x² + x
- ( - 2x² + 4x )
- 3x + 6
- ( - 3x + 6 )
0
Ergebnis verifiziert: Weil der finale Rest genau null ist, wissen wir, dass der Rechenweg vollständig aufgegangen ist. Das verbleibende reduzierte Polynom lautet x² - 2x - 3.

Nach der Gradreduzierung setzen wir den gewonnenen Quotienten gleich null, um die verbleibenden Nullstellen mit der bekannten pq-Formel zu lösen:

x² - 2x - 3 = 0
Anwendung der pq-Formel mit p = -2 und q = -3:
x2,3 =
-(-2) ± √[ (-2)² - 4 · 1 · (-3) ] 2 · 1
x2,3 =
2 ± √[ 4 + 12 ] 2
x2,3 =
2 ± 4 2
x₂ = 3 x₃ = -1

Die vollständige Lösungsmenge der kubischen Ausgangsfunktion lautet somit: L = {-1; 2; 3} (inklusive der anfangs geratenen Nullstelle x = 2).
